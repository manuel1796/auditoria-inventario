# auditoria-inventario
Sistema en Python que analiza el stock de productos tecnológicos y genera un informe de reabastecimiento, identificando qué artículos necesitan pedido y en qué cantidad.

Este programa recorre una lista de productos, compara el stock actual contra el stock mínimo requerido, y produce:

Una tabla resumen con el estado de todos los artículos.
Una lista de pedidos con los artículos que necesitan reabasto.

Desarrollado como ejercicio de programación en Python con matrices y funciones.
